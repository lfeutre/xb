## Installation

Just add it to your ``rebar.config`` deps:

```erlang
  {deps, [
    ...
    {xb, ".*",
      {git, "git@github.com:YOURNAME/xb.git", "master"}}
      ]}.
```

And then do the usual:

```bash
    $ rebar get-deps
    $ rebar compile
```
